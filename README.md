# Pinnacle-Adventures

Pinnacle Adventures is a challenging platformer where players navigate through levels using critical thinking and reaction skills. The game features various platforming obstacles, requiring the player to use precise controls to reach the end of each level.

Controls:
- A/Left Arrow: Move left
- D/Right Arrow: Move right
- Spacebar: Jump

Objective:
- Reach the end of each level while overcoming difficult jumps and hazards.
- Players have limited lives to complete each level, and once they succeed, they can spawn from checkpoints.

Game Features:
- Medieval-inspired pixel art world.
- Multiple courses and pathways with environmental challenges.

Game Story:
Pinnacle Adventures follows the journey of a hero exploring the world of Pinnacle, a place filled with challenging platforming sections and mysteries. The name "Pinnacle" signifies the hero's ultimate goal of reaching the peak of their journey.
